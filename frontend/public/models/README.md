Download a GLB/GLTF model and place it here as `f1.glb`.

Recommended sources:
- Sketchfab (search "Formula 1 glb", "F1 car gltf")
- CGTrader (filter for glTF/GLB)

Choose a model with:
- realistic wheels, aerodynamic wings, cockpit details
- dark / carbon materials
- .glb or .gltf format (preferred)

Avoid FBX, ultra high-poly movie assets, or broken textures.

Place the file at:
`/public/models/f1.glb`

Example usage in code: load `/models/f1.glb` with `useGLTF('/models/f1.glb')`.
